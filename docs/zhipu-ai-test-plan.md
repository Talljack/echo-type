# EchoType ZhiPu AI (GLM) 功能测试文档

## 测试目标

将 EchoType 应用切换到 Z.AI (智谱) 作为 AI 提供商，使用 `.env.local` 中配置的 `ZAI_API_KEY`，全面测试所有 AI 相关功能。

## 测试环境

- **提供商**: ZhiPu AI (GLM)
- **模型**: GLM-4 Plus (默认), GLM-4 Air, GLM-4 Flash, GLM-Z1 Flash
- **API Base URL**: `https://open.bigmodel.cn/api/paas/v4`
- **API Key**: 通过 `.env.local` 的 `ZAI_API_KEY` 或设置页面手动输入
- **应用 URL**: `http://localhost:3000`

---

## 测试用例

### TC-01: 设置页面 - 切换到 ZhiPu AI 提供商

**路径**: `/settings`

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 打开设置页面 | 显示 AI Provider 配置区域 |
| 2 | 在 Provider 下拉菜单中找到 "Chinese" 分组 | 能看到 Zhipu AI (GLM) 选项 |
| 3 | 选择 "Zhipu AI (GLM)" | 显示 ZhiPu 配置面板，蓝色主题 |
| 4 | 查看 API Key 输入框 | 有占位符提示 |
| 5 | 输入 API Key 或使用 env 默认值 | API Key 显示为密码模式 |
| 6 | 点击 "Connect" 按钮 | 连接成功，显示绿色状态 |
| 7 | 确认 ZhiPu 被设为 Active Provider | 显示绿色活跃标识 |

### TC-02: 模型选择与切换

**路径**: `/settings`

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 查看模型选择器 | 显示 GLM-4 Plus (默认选中) |
| 2 | 切换到 GLM-4 Air | 模型切换成功 |
| 3 | 切换到 GLM-4 Flash | 模型切换成功 |
| 4 | 切换到 GLM-Z1 Flash | 模型切换成功 |
| 5 | 切回 GLM-4 Plus | 确认默认模型工作正常 |

### TC-03: AI 聊天功能 (Chat)

**路径**: 任意应用页面 → 右下角 Chat FAB

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 点击右下角聊天浮动按钮 | 聊天面板滑出 |
| 2 | 查看提供商标识 | 显示 "Zhipu AI" 或类似标识 |
| 3 | 发送消息 "Hello, can you help me learn English?" | 收到流式回复，内容与英语学习相关 |
| 4 | 发送消息 "What's the difference between 'affect' and 'effect'?" | 收到准确的英语解释 |
| 5 | 检查回复格式 | Markdown 正确渲染 |

### TC-04: 翻译功能 (Translation)

**路径**: `/listen/[id]`, `/speak/[id]`, `/write/[id]`

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 打开一个 Listen 练习页面 | 页面正常加载 |
| 2 | 开启翻译显示 | 翻译按钮可用 |
| 3 | 等待翻译加载 | 显示逐句翻译结果 |
| 4 | 切换目标语言为中文 | 显示中文翻译 |
| 5 | 切换目标语言为日文 | 显示日文翻译 |
| 6 | 在 Speak 页面测试翻译 | 同样正常显示 |
| 7 | 在 Write 页面测试翻译 | 同样正常显示 |

### TC-05: AI 推荐功能 (Recommendations)

**路径**: `/listen/[id]`, `/speak/[id]`, `/write/[id]`

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 确认设置中推荐功能已开启 | 推荐数量设为 3/5/10 |
| 2 | 打开一个练习页面 | 页面正常加载 |
| 3 | 等待推荐面板加载 | 底部显示推荐卡片 |
| 4 | 检查推荐内容 | 包含标题、描述、关系类型 |

### TC-06: AI 内容生成 (AI Generate)

**路径**: `/tools` → AI Generate 标签

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 打开 Tools 页面 | 显示工具标签页 |
| 2 | 切换到 "AI Generate" 标签 | 显示生成界面 |
| 3 | 选择类型: Words, 主题: Travel, 难度: Beginner | 参数设置成功 |
| 4 | 点击生成 | 返回与旅行相关的单词列表 |
| 5 | 选择类型: Sentences, 主题: Technology | 返回科技相关句子 |
| 6 | 选择类型: Articles, 主题: Business | 返回商务相关文章 |
